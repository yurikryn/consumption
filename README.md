 Обліковець лічильників

Це офлайновий html-js-app, 
який дозволяє вести облік різних квартирних лічильників по
різних адресах.
У ньому купа неочивидних можливостей, але оскільки він створювався для власного використання,
то help-а немає :(, а тут його писати мені лінь ( адже невідомо, чи це комусь ще потрібно).

Як запустити:
- скачати в якусь папку
- слідувати логіці запитів
- коли попросить зберегти - зберігати в цій папці під іменем "main.txt" - і тільки таким,
  бо тоді після перезапуску прочитає його. 
Якщо хтось попросить - допишу help, знищу одного жука та покращу логіку.
